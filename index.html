Questionnaire app

It is a Web application to collect data from people then calculate the data using K-means Clustering algorithm. This application is built using React.js, Express.js and data stored in MySQL database.

Road condition detection app

An android application to show users which roads have potholes so that users can be careful passing through these roads. This app is built using React Native, Express.js and Arduino. The data collection tool is made using nodemcu as a microcontroller, a GPS module to get the current location and a gyroscope and accelerometer module to record shocks when the vehicle passes through potholes on the road.
